# Accounts Payable User Guide

## Overview
Manage vendor relationships, track bills, and process payments.

## Managing Vendors

### Add Vendor
1. **AP > Vendors > Add Vendor**
2. Enter: Code, Name, Email, Phone, Payment Terms
3. **Save**

### Edit Vendor
1. Find vendor in list
2. Click **Edit**
3. Update details
4. **Save**

## Processing Bills

### Create Bill
1. **AP > Bills > Add Bill**
2. Select vendor
3. Enter: Invoice #, Date, Due Date
4. Add line items
5. **Save**

### Approve Bill
1. Open bill
2. Review details
3. Click **Approve**

## Making Payments

### Single Payment
1. **AP > Payments > Make Payment**
2. Select vendor and bills
3. Enter payment details
4. **Process Payment**

### Batch Payments
1. **AP > Batch Payments**
2. Select multiple bills
3. Set payment date and method
4. **Process Batch**

## Reports

### AP Aging
**AP > Reports > AP Aging**
- View bills by age (Current, 30, 60, 90+ days)
- Export to Excel/PDF

### Vendor Statement
**AP > Reports > Vendor Statement**
- Select vendor and date range
- View all transactions

### 1099 Forms
**AP > 1099 Forms**
- Select tax year
- Generate forms
- Print or e-file

## Tips
- Pay bills on time for discounts
- Use batch payments for efficiency
- Review aging report weekly
- Keep vendor info updated
