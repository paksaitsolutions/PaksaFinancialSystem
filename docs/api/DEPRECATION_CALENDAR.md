# API Deprecation Calendar

This calendar tracks announced deprecations, their timelines, and removal targets. All dates are UTC.

| Item | Deprecation Announced | Removal Target | Notes |
| --- | --- | --- | --- |
| *(None)* | — | — | No deprecations currently scheduled. |

## How to Add an Entry
1. Add the endpoint or field slated for deprecation.
2. Provide the announcement date and minimum 90-day removal target.
3. Include migration guidance and contact owners.
